1. Html stands for hypertext markup language.It is standard markup language used to create web page.
Html is a combination of hypertext and markup.Hyper text is used to define the link between the web page.
Markup language is used to define the text document within the tag to define the structure of the webpage.
This  language is used to annotate the text to make the computer understand the language.

It is platform independent
